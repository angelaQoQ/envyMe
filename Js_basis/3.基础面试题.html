# 4. 传值调用(call by value) 修改为传名调用(call by name)

这两个概念是对函数调用时参数处理的时机选择

传值调用: 函数调用之前把实参表达式的值求出来. C语言采用
    缺点: 如果存在没有使用到的参数,就会损耗性能

传名调用: 函数调用时,用到哪个参数再获取这个参数的值. Haskell采用

传值改传名 : 把实参列表修改为函数callback调用, 将实参求值操作放在callback声明中

function fn( a,b ){
    console.log(a+b)
}

let x = 1
fn( x * 2 + 3 , 100 )

// 把fn调用改为传名调用

function thunk(){
    let x = 1
    return [x*2+3 , 100]
}
fn(...thunk())
